MATLAB functions and simulators for deterministic modeling and prediction of time series using signal decomposition into trend, seasonal component, and residual, as well as 
for extending the deterministic framework with stochastic modeling of residual components using Autoregressive (AR) and Autoregressive Moving Average (ARMA) processes.
Comparative analysis of AR and ARMA models were performed, highlighting the trade-off between computational efficiency and prediction accuracy across different types of time series.
